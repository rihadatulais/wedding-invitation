# Frontend Wedding Invitation

Frontend React + Vite untuk gallery template, preview template, dan halaman undangan tamu.

## Menjalankan project

```bash
npm install
npm run dev
```

## Cara ganti foto sementara (sebelum portal admin)

1. Copy file foto ke `frontend/public/photos/`
2. Rename foto background hero menjadi `hero.jpg`
3. Jika perlu foto mempelai pria, letakkan juga file `groom.jpg`
4. Jalankan atau refresh `npm run dev` dan foto akan langsung tampil di browser tanpa perlu restart server
